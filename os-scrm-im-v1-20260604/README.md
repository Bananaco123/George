# OS-SCRM IM V1 Demo · 2026-06-04

## 定位

本目录是 OS-SCRM WhatsApp Web IM 的独立 V1 发布包，不修改或覆盖仓库根目录中的已有项目。

## 启动

```bash
npm install
npm run dev
```

默认端口：`5181`

打开路径：

```text
http://localhost:5181/scrm
```

如未登录，会先进入登录页；使用默认演示账号登录后自动回到 `/scrm`。

## 构建

```bash
npm run build
```

构建产物位于 `dist/`，已使用相对资源路径，可作为独立静态目录发布。

## 演示重点

- 多 WhatsApp 账号切换
- 非当前账号红点、未读数、闪烁
- 当前账号会话列表与标签组切换
- 单聊 / 群聊消息流
- 消息发送、送达、已读和失败状态
- 消息编辑、已编辑标识、双方撤回和仅我们删除
- 群聊 `@` 成员、引用、表情回应、转发和失败消息重发
- 语音播放动效和附件预览
- 手动建群、群成员管理、入群审批
- 当前账号掉线遮罩与重新扫码登录
